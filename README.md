# MaatiTrace

MaatiTrace is organized as a geospatial land intelligence platform with backend services, data pipelines, infrastructure, notebooks, and a frontend workspace.

## Structure

- `backend/services/` - deployable backend services.
- `backend/platform/` - shared platform components such as config, logging, storage, security, and observability.
- `backend/pipelines/` - ingestion, raster, preprocessing, feature, training, and inference workflows.
- `backend/notebooks/` - research and experimentation notebooks.
- `backend/infrastructure/` - Docker, compose, Render, GitHub Actions, and Terraform assets.
- `frontend/` - frontend application workspace.
