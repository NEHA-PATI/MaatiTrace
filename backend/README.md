# MaatiTrace Backend

Backend workspace for MaatiTrace services, shared platform modules, pipelines, notebooks, infrastructure, scripts, tests, and documentation.

## Layout

- `services/` - API gateway and domain services.
- `platform/` - shared runtime and infrastructure helpers.
- `pipelines/` - data and ML pipeline code.
- `notebooks/` - exploratory analysis and model research.
- `infrastructure/` - deployment and provisioning assets.
- `scripts/` - operational and developer scripts.
- `tests/` - backend test suite.
- `docs/` - backend documentation.
